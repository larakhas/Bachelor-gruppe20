URL-shortener

This bachelor's thesis, in collaboration with nav, deals with the development of an internal URL shortening service for secure and efficient link sharing. The solution is built with Kotlin and React, integrated into nav's Test Nais platform, and allows users to manage links with a focus on security and GDPR compliance.
